# IntelliJ Idea CWL plugin

## Description

Plugin for IntelliJ IDEA to work with
[Common Workflow Language](http://www.commonwl.org/v1.0/index.html) files

## Requirements

To compile you will need:

* IDEA 2017.1.2
* Kotlin plugin
* JDK 8
* IntelliJ IDEA Plugin SDK
